Hosting a Run

We're looking for places to run The Labor Wars! In order to host a game, we need a house capable of seating 20 people for a weekend, plus space to cook and serve a meal.

If you would like to host a run of The Labor Wars, please let Nat Budin know. And thanks!

June 18-20, 2010 - Belmont, Massachusetts

The first run of The Labor Wars was in Belmont, Massachusetts the weekend of June 18-20, 2010. The game was hosted in a private residence.

The deadline to sign up for this game has now passed. Stay tuned for more information on future runs.
